# 来源映射与使用边界

本 Skill 的工程化规则来自以下资料类型：

| 资料类型 | 用途 | 使用方式 |
|---|---|---|
| CiA402 / IEC 61800-7 | 模式、对象字典、Controlword/Statusword、FSA 基本语义 | 只做摘要和工程化转述 |
| ETG.6010 | EtherCAT CiA402 伺服驱动共同实现行为、同步、PDO、Device Type | 用于实现约束和排障优先级 |
| ETG.1000 | EtherCAT 物理层、数据链路层、应用层、CoE、FoE、EoE、ESM | 用于分层诊断和对象/协议归因 |
| AN ET9300 / SSC | SSC 工程结构、对象字典生成、PDO Mapping、TwinCAT 配置 | 用于代码审查和实现建议 |
| 厂商应用手册 | TwinCAT 页面、同步模式、PDO 例子、设备对象行为 | 用于联调步骤和现象解释 |
| 项目实测经验 | 0x6060/0x6061、statusword bit12、多轴 WcState、共享内存 | 用于优先排查路径 |
| EtherCAT 网页知识库 | 示例 SDK、片上 ESC 外设、ECAT_CIA402/ECAT_IO 的实现细节、PHY/EEPROM/中断/MDP/动态 PDO 实例 | 只做工程实现参考和代码排查提示，不替代规范 |

## 版权和保密边界

- 不要复制标准文档原文或大段表格。
- 不要把受限 ETG/CiA PDF 打包进对外 skill。
- 输出时使用“根据某类规范/手册/工程经验”的概括性描述。
- 用户需要精确条款时，引导其查看合法获得的原始文档。

## 信息优先级

当资料冲突时，按以下优先级判断：

1. 用户当前项目代码、ESI、TwinCAT 实测日志。
2. 目标芯片/目标 SDK 与实际从站栈限制。
3. ETG.6010/ETG.1000/CiA402 规范语义。
4. 第三方应用手册和示例。
5. 经验性推断。

输出中必须明确哪些是事实，哪些是推断。



## 网页知识库使用边界

- 网页知识库资料用于回答示例 SDK、片上 ESC 外设、ECAT_CIA402/ECAT_IO 例程、Flash 模拟 EEPROM、ESC_CTR/IO_CFG、PHY Link 配置等工程实现问题。
- 若用户没有使用同类芯片或同类 SDK，不要默认套用网页示例中的寄存器宏、函数名或硬件连接方式；只可提取通用思想。
- 需要网页截图、寄存器图或完整代码时，不在 Skill 内复刻原文；应以用户提供的资料或项目代码为准。
