# 电机控制固件

默认使用中文回答和生成说明。仅在用户明确要求英文，或项目注释/接口规范要求英文时使用英文。

使用此 skill 处理嵌入式电机控制设计、实现、调试和评审。数值/控制细节结合 `dsp`，C 代码结合 `c99-standard-c`，评审输出结合 `review`。

## 工作流程

1. 确认电机类型、逆变器拓扑、电流采样、位置反馈、PWM 频率、控制环周期、ADC 触发时序和安全要求。
2. 将被控对象/控制数学与硬件驱动和调度胶水分离。
3. 明确单位和数值格式：A、V、rpm、rad/s、电角度、机械角度、Q 格式和饱和限制。
4. 定义启动、校准、对齐、开环、闭环、故障和停止状态。
5. 分阶段验证投产：空载检查、电流环验证、速度环验证、保护触发和长稳稳定性。

## 检查项

- 电流/电压/温度/位置故障必须强制进入安全状态。
- PWM 更新、ADC 采样、编码器捕获和控制环时序必须确定。
- PI/PID anti-windup、输出饱和和积分器复位条件必须明确。
- 角度 wrap、坐标变换和符号约定必须测试。
- EtherCAT 或其他现场总线 setpoint 在命令扭矩/速度/位置前必须做超时和合理性检查。
