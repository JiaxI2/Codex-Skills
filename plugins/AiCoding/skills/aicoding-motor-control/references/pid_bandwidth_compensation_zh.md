# PID、三环带宽、前馈与补偿

## 1. 三环带宽

```text
电流环：最高带宽，直接受 PWM/ADC/计算延迟限制。
速度环：低于电流环，避免把电流噪声和速度估算噪声放大。
位置环：低于速度环，避免机械共振和多轴同步抖动。
```

推荐起点：

```text
速度环带宽 <= 电流环带宽的 1/5 ~ 1/10。
位置环带宽 <= 速度环带宽的 1/3 ~ 1/10。
实际值必须由阶跃、频响、负载和噪声测试确定。
```

## 2. 电流环 PI

必须有：

```text
积分限幅
输出限幅
anti-windup
Vbus 限幅
Id/Iq 独立状态
饱和状态输出
```

## 3. 速度/位置 PID

```text
D 项必须滤波。
低速速度噪声大时，优先 PI + 速度滤波/observer。
位置环输出速度或转矩时必须有斜率限制。
三环前馈不能绕过保护限幅。
```

## 4. 何时做补偿

```text
高速：BEMF、弱磁、Vbus。
大电流：dq 交叉耦合。
低速爬行：摩擦、死区、offset、编码器误差。
机械臂/机器狗：重力补偿、摩擦补偿、速度/加速度前馈。
多轴联动：同步 tick、轨迹前馈、插补一致性。
```

## 5. 测试

```text
电流阶跃：看 Id/Iq 超调、建立时间、饱和。
速度阶跃：看速度响应、Iq 限幅、过零抖动。
位置阶跃：看跟随误差、机械振动、末端抖动。
扫频/SFRA：看带宽、相位裕度、共振点。
负载扰动：看恢复时间和积分饱和。
```
