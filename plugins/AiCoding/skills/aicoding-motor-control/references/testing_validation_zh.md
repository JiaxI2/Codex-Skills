# 电机控制测试验证

## 1. 最小 Bring-up

```text
相序确认
ADC offset
电流方向确认
编码器方向/零位/极对数
开环 Vq
闭环 Id/Iq
速度闭环
位置闭环
```

## 2. 控制测试

```text
电流阶跃
速度阶跃
位置阶跃
正反转过零
低速爬行
高速限压
负载扰动
急停/抱闸联动
多轴同步轨迹
```

## 3. 故障注入

```text
过流
过压/欠压
过温
编码器断线/CRC 错误
ADC 异常
PWM overrun
通信超时
同步丢失
```

## 4. 通过标准

```text
无不可解释 overrun。
故障进入确定安全状态。
保护动作可复现。
参数调试过程有 trace。
重构前后波形一致或差异可解释。
```
