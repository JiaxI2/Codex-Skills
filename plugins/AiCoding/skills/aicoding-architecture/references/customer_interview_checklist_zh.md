# 客户访谈清单

## 1. 主控/板卡

```text
当前 MCU/MPU/DSP/FPGA 型号？
是否固定板卡？是否未来换主控？
是否使用厂商 SDK？版本？
是否有 RTOS/Linux？
是否多核？核间通信方式？
```

## 2. 是否参考仓库复刻

如果主控匹配参考项目，询问：

```text
是否希望参考该仓库的目录结构？
是否希望复刻参数系统？
是否希望复刻日志/黑匣子？
是否希望复刻 Bootloader/升级方式？
是否希望复刻通信协议？
是否只需要吸收思想，不需要实现？
```

## 3. 量产

```text
是否量产？
升级通道？
是否需要签名？
是否需要安全认证？
产线如何烧录？
售后如何定位问题？
```
