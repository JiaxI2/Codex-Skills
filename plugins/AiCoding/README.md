# AiCoding Codex Plugin

AiCoding is the deployable Codex kit for JiaxI2 embedded development. It packages selected generated skills, Git-Skill repository governance, and Codex lifecycle hooks in one Codex plugin layout.

## Layout

```text
AiCoding/
├── .codex-plugin/
│   └── plugin.json
├── skills/                 # generated by scripts/build-plugin.ps1
├── hooks/
│   ├── hooks.json
│   └── git_governance_stop.py
├── BUILDINFO.json          # generated by scripts/build-plugin.ps1
└── README.md
```

Only `.codex-plugin/plugin.json` is inside `.codex-plugin/`. Skills and hooks stay at the plugin root.

## Build

Run from the Codex-Skills repository root:

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File scripts/build-plugin.ps1 -Plugin AiCoding -Configuration Development -Clean -Verify
```

Do not hand-edit `skills/`. The source mapping is `config/aicoding-plugin-pack.json`.
